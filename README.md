# Restaurant Reviews Sentiment Analysis

In this project, we will conduct sentiment analysis on restaurant reviews. We will analyze a dataset of restaurant reviews to classify them into positive, negative, or neutral sentiments. Additionally, we will explore the reasons behind negative sentiments in the reviews. This comprehensive analysis will provide valuable insights into customer experiences and restaurant performance.
